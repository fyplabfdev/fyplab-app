## Overview

Fyplab transforms short-form video consumption into an active prediction experience.  
Instead of passive scrolling, users analyze engagement signals and predict whether a video will become **VIRAL** or **NOT VIRAL**.

This repository contains the **frontend application** currently running in production (beta).

---

## Key Features

- Vertical short-video feed (mobile-first)
- VIRAL / NOT VIRAL prediction interface
- Point-based staking and reward system
- Real-time engagement metrics
- User profiles and prediction history
- Trending indicators
- Creator discovery experience

---

## Tech Stack

- **Framework:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** TanStack Query
- **Routing:** Lightweight client routing
- **API:** REST-based backend service
